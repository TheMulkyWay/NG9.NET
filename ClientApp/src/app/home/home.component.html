Full Stack Microservice Web Apps

For organisations needing more Web components delivered more quickly this a small free example showing how this might be achieved.

This means any operations requiring, secure data gathering and sharing can be up and running in less time, and that this is achievable because the means of production is cloud based.

As a practical example using the simple functionality here I am prepaired to demonstrate feature changes together with the build pipeline and live deployment to show how well any change management can meet your requirments.
